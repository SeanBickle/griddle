What is griddle?
griddle is a daily word search.

There is a new griddle every day, and everyone griddles the same griddle.

This project is inspired by Wordle and was originally created during a windy weekend in February, 2022.
How to play
  • Tap adjacent tiles to select a word
  • Earn points for each tile in the word
  • Earn 2x points for words with 7+ letters
  • Enter up to five words a day and aim to achieve the highest possible score.
